Monaca provides several third-party plugins by default, but you can also import other third-party plugins or your own plugin to Monaca project.We also provide a way to extend your program using native code. Please refer to :ref:`custom_cordova_plugin` on how to create and use a native plugin.The followings are pre-configured third-party Cordova plugins provided by Monaca:.. toctree::  :maxdepth: 2  datepicker  barcode_scanner  share  webintent  html5_resource_encryption  secure_storage  in-app_updater  sqlcipher
