===============================
Monaca Backend
===============================

There is a growing need for mobile apps to have user management functions which allow users to register and keep their data in a database. Monaca Backend provides these functionality in advance as a cloud-based service that can manage users without a need for server-side development on the client part.

*  Monaca Backend sample app is described in :ref:`backend_database_memo`.
*  Monaca Backend API is listed in :ref:`backend_api_index`.
